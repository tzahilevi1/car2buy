# מדריך קידום אורגני — Car2Buy

מדריך מעשי לצעדים שצריך לבצע **ידנית** (דורשים התחברות עם החשבון שלך). כל שיפורי הקוד באתר כבר בוצעו.

הכתובת החיה: **https://tzahilevi1.github.io/car2buy/**

---

## 1. Google Search Console (חובה — כך גוגל "רואה" את האתר)

זה הכלי שמאפשר לגוגל לאנדקס את האתר ומראה לך מילות חיפוש, קליקים ובעיות.

1. היכנס ל-https://search.google.com/search-console עם חשבון Google.
2. לחץ **Add property** → בחר **URL prefix** → הדבק: `https://tzahilevi1.github.io/car2buy/`
3. **אימות בעלות** — הדרך הכי קלה כאן היא **HTML tag**:
   - גוגל ייתן לך תגית בסגנון `<meta name="google-site-verification" content="XXXXX">`.
   - שלח לי את הקוד (ה-`content`) ואני מזריק אותו לכל עמודי האתר, נעשה push, ואז תלחץ **Verify**.
4. אחרי אימות — **Sitemaps** בתפריט → הוסף: `sitemap.xml` → Submit.
   (ה-sitemap כבר קיים: https://tzahilevi1.github.io/car2buy/sitemap.xml)
5. **URL Inspection** → הדבק את כתובת הבית → **Request Indexing** כדי לזרז אינדוקס.

> הערה: אינדוקס מלא לוקח מ-כמה ימים עד שבועות. זה נורמלי.

## 2. Google Business Profile — "גוגל לעסק שלי" (קריטי לקידום מקומי בישראל)

זה מה שמציג את העסק במפות Google ובחיפושים מקומיים ("ליסינג רכב ליד…"). **הכי משתלם עבור עסק ישראלי.**

1. היכנס ל-https://www.google.com/business עם חשבון Google.
2. **Add your business** → שם העסק: **Car2Buy**.
3. קטגוריה: **סוכנות רכב / מימון רכב** (Car dealer / Auto broker).
4. כתובת: אם יש משרד פיזי — הזן אותו (יאפשר הופעה במפות). אם עסק אונליין ללא קבלת קהל — סמן "אני מספק שירות באזור" ובחר אזורי שירות.
5. טלפון + כתובת האתר: `https://tzahilevi1.github.io/car2buy/`
6. **אימות** — בדרך כלל בגלויה בדואר (קוד), לפעמים בטלפון/וידאו.
7. אחרי אימות: הוסף **תמונות** (לוגו, רכבים, לקוחות מרוצים), **שעות פעילות**, ותיאור עסק. עודד לקוחות להשאיר **ביקורות** — זה הגורם מספר 1 בדירוג מקומי.

> ⚠️ **חשוב לעקביות (NAP):** השם, הטלפון והכתובת ב-Google Business חייבים להיות **זהים בדיוק** למה שמופיע באתר. ברגע שתשלח לי את פרטי העסק האמיתיים, אעדכן אותם בכל האתר כדי שיהיה תואם.

## 3. אנליטיקס (מומלץ — למדוד תנועה)

האתר כבר מכין `dataLayer` (תשתית ל-Google Tag Manager / GA4). כדי להפעיל מדידה:
- פתח חשבון **Google Analytics 4** (https://analytics.google.com), קבל **Measurement ID** (בסגנון `G-XXXXXXX`).
- שלח לי אותו ואחבר אותו לאתר (הזרקה מרוכזת ב-`site.js`, כולל Consent Mode לבאנר העוגיות הקיים).

## 4. מה שכבר בוצע בקוד (רקע)
- ⚡ תמונות: 136MB → 17MB (מהירות טעינה = גורם דירוג).
- 🏷️ כותרות ותיאורי מטא ייחודיים לכל עמוד + Open Graph + Twitter + canonical.
- 🗂️ `sitemap.xml`, `robots.txt`, favicon, manifest.
- 🧩 נתונים מובנים (schema): WebSite + SearchAction, AutoDealer, Vehicle, FAQ, Article.
